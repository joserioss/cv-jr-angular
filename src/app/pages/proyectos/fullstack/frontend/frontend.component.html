Front-end

Por el lado FRONT-END manejo HTML5, CSS3 y JavaScripts desde septiembre del 2019, además de técnicas responsivas y el uso de jQuery. Además, manejo el framework Angular con el lenguaje de programación TypeScripts, incluyendo a angular material y bootstrap, además de angular routing.

Tecnologías usadas

Proyectos con Angular

language
CLINIC-APP https://github.com/joserioss/clinicapp-frontend Proyecto en desarrollo trabajado con angular, angular material y angular routing. Se comunica con el back-end mediante un servicio REST, por lo que en el front-end también esta modelado el CRUD básico para acceder a los datos almacenados.
language
COORDENADAS-GPS https://github.com/joserioss/cordenadas-gps-frontend Proyecto Angular que permite ingresar una imagen o grupo de imágenes, para almacenar en una carpeta local del proyecto y en una base de datos. De estas imágenes se adquieren los metadatos GPS para poder mostrar con un marcador en un mapa del mundo, el lugar en donde se tomó la fotografia, mediante el uso de AGM (angular google map).
