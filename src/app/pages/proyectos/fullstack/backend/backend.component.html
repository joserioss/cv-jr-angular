BACK-END

Mi fuerte en el BACK-END se centra principalmente en el desarrollo de aplicaciones JEE con Spring, para una arquitectura REST, tal como se puede ver en los siguientes proyectos:

language
CLINIC-APP https://github.com/joserioss/clinicapp-backend Proyecto en desarrollo que consta de las entidades: paciente, medico, especialidad, diagnóstico, tratamiento y consultas médicas que recopila todas las entidades anteriores, incorporando un CRUD para cada una de estas entidades y el acceso a su información es mediante un servicio REST que se comunica con un Front de Angular.
language
PROYECTO-IOT https://github.com/joserioss/proyectoIoT Proyecto del tipo JSP desarrollado para el cierre del bootcamp de Talento Digital, consta de las entidades: usuario, dispositivo y sensor, en donde se puede observar en tiempo real la lectura de sensores que están conectados y registrados en el sistema. Para la recepción de los datos, se hace uso de un servicio REST que recibe la información mediante un método POST, y la almacena en una base de datos para su posterior adquisición. Por otro lado, en este proyecto se implementa Spring Security.

Tecnologías usadas

En el siguiente tab-group indicare un resumen de las tecnologías en las que he trabajado para diferentes back-end:

  • Java
  • Spring
  • Hibernate
  • JWT
  • Rest
  • MVC
  • JUnit 5
  • JHipster

Con Java aprendí el paradigma orientado a objetos, inicialmente usando Java 8, y luego el desarrollo de aplicaciones web dinámicas con java servlets y JSP. Además de hacer uso de JDBC como interfaz de conexión a las bases de datos mySQL y postgreesql.

Experiencia en el desarrollo de aplicaciones JEE con Spring Framework, haciendo uso de spring boot, hibernate para la persistencia de datos, modelo-vista-controlador, Spring Security, JWT, servicios REST. Tambien trabaje en pruebas unitarias con JUnit 5, usando Fake y Mock.
